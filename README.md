**สิ่งที่เว็บนี้ไม่มี:**  
- Usable:ใช้งานง่าย  
  - ยากต่อการใช้งาน bgสีฉูดฉาดเกินไป ทำให้อ่านตัวหนังสือยาก  
- Credible: น่าเชื่อถือ  
  - เว็บไม่สวย ดูไม่น่าเชื่อถือเหมือนเว็บปลอม  
- Desirable: น่าใช้
  - ไม่น่าใช้ เหมือนhtml ที่ไม่ใช้css
